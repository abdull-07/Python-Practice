## ğŸš€ Getting Started

### Prerequisites

- **Python 3.8 or higher** installed on your system
- A code editor (VS Code, PyCharm, Sublime Text, etc.)
- Basic command line knowledge

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/abdull-07/Python-Practice.git
   cd Python-Practice
   ```

2. **Verify Python installation:**
   ```bash
   python --version
   ```

3. **Run your first exercise:**
   ```bash
   # Windows (PowerShell)
   python "Basic\Level 1 Basic Python Practice (Warm-up)\1_ greeting.py"
   
   # Windows (Command Prompt)
   python Basic\Level 1 Basic Python Practice (Warm-up)\1_ greeting.py
   
   # Linux/Mac
   python3 Basic/Level\ 1\ Basic\ Python\ Practice\ \(Warm-up\)/1_\ greeting.py
   ```
